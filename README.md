# Log Analyzer

A resilient CLI-based log ingestion and analysis tool that parses mixed-format server logs, handles malformed lines gracefully, and produces operational summaries.

## Features

- Parses multiple timestamp formats
- Handles malformed lines safely
- Supports JSON log lines
- Streams large files efficiently
- Detects slow endpoints
- Reports status code distribution
- Identifies suspicious IPs
- Generates anomaly summaries
